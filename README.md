# Selenium-Java-Hybrid-Automation-Framework👨‍💻

## How to run
run
```bash
run.bat 
```
file <br>

--------------------
type
```bash
"mvn clean install"
```
in terminal

## Tech Stack and main features
✅ Selenium4<br>
✅ Java<br>
✅ TextNG<br>
✅ page object model<br>
✅ log4j2<br>
✅ AutoIT<br>
✅ Extent reports<br>
✅ TestNG RetryAnalyzer<br>
✅ Data-Driven testing using TestNG DataProvider<br>
✅ Multiple browser testing<br>
